* components
 - 실제 vue 코드들
````
<template> - html
<script> - js 코드
<style> - css
````
* router
 - urls와 비슷
```
import Vue from 'vue'
import Router from 'vue-router'
import HelloWorld from '@/components/HelloWorld'
 
Vue.use(Router)
export default new Router({
  routes: [
    {
      path: '/',
      name: 'Hello',
      component: HelloWorld
    }
  ]
})
```
